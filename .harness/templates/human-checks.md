# Human Checks

本文件记录 Unity 编辑器、Prefab、真机、视觉表现等 AI 不能可靠自行完成的检查。

状态可选：

- `pending`
- `passed`
- `failed`
- `waived`

## 检查项

| 状态 | 检查项 | 操作者 | 日期 | 证据或备注 |
| --- | --- | --- | --- | --- |
| pending | 在 Unity 编辑器中确认 Inspector 配置无误。 |  |  |  |
| pending | 确认相关 Prefab 不存在 Missing Script。 |  |  |  |
| pending | 进入 Play 模式确认目标场景可运行。 |  |  |  |
